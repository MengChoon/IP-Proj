EasyShop
This is an ecommerce website, featuring a home, product, about, contact and account page. Based on my shopping experience on the web, i came to the conclusion that my website should have the core of a shopping website. a login page, always linking the users account for personalised products based on prefrences. an all products page to show all the products. and a reccomended section on the homepage.


Design process
This website is for people who are interested in buying items online. it is the best for people who want to buy exclusive items from our shopswith the badges they have (shown in adobe XD for AD).

feature
feature 1) in the home page, the navigation buttons go to different sites when pressed. 
feature 2) navigation cart shows items currently in cart
feature 3) responsive webpage
feature 4) signup and login details sent by api to restdb 
feature 5) login information saved to local localStorage
feature 6) account page shows the user name and a pic after logged in. option to sign out available and after signed out the account page links back to login page
feature 7) contact us section links to restdb api 
feature 7) message will be collected in restdb with input of name, message, email and number 
feature 8) for login and contact, there will be a message thay tells you if message has een sent or if it hasnt
feature 9) there is an individual product page where you can add the item to cart. item saves to local storage and displays in cart page.
feature 10) cart page adds the total item price and shows the subtotal inclusive of tax (assuming 8% like sg :'( )


Features Left to Implement:
